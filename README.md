fgcsgdm
=======

Ground Control Station for Gas Distribution Mapping written in FLTK

Compilation:
(1) In the TOP dir, execute the following commands:
    mkdir build
    cd build
    cmake ..
    make
(2) 

Dependencies:
(1) OpenGL library

Note:
(1)The project directory is fgcsgdm, while other folders (include src) are FLTK's.
