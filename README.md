RAOS -- Simulation Framework for Robot Active Olfaction
=======

Compilation:
----
(1) In the TOP dir, execute the following commands:<br>
    mkdir build<br>
    cd build<br>
    cmake ..<br>
    make<br>
(2) <br>

Dependencies:
----
(1) OpenGL library<br>
(2) boost
(3) make, cmake, gcc, g++, gfortran
(4) for rotor wake calculation:
    cuda toolkit, and a graphic card
